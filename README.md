# 2D Graphics Editor in C

## Description

A menu-driven 2D Graphics Editor implemented in C using a 2D character array as a drawing canvas.

## Features

* Draw Rectangle
* Draw Circle
* Draw Line
* Draw Triangle
* Display Canvas
* Menu Driven Interface
* User Input for Shape Parameters

## Technologies Used

* C Programming
* Character Arrays
* Functions
* Menu Driven Programming

## Compilation

gcc main.c -o editor -lm

## Execution

./editor
